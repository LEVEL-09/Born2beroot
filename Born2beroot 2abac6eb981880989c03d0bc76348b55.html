What is virtualization?

Virtualization is the process of creating virtual machines (gust operating system) in physical machine (host), use hypervisor (program or firmware) to allocate hardware resources-such Memory, cpu, Storage, Network to can operation systems shear this resources.

What is type hypervisor?

They are two main type:

  • Bare-matel (type 1):

    In this type we have hypervisor in hardware without any OS layer between them so it can access directory to resources.

  • Hosted (type 2):

    In this type we have hypervisor in OS so we have OS layer between hardware and hypervisor.

What is KDump

It's a kernel feature for Linux system administrators that captures kernel crashes and saves the contents of the crashed kernel memory in a file called Core Dump (aka vmcore). It then calls Kexec (a system call) to use a secondary kernel without a full reboot.

Difference between BIOS and UEFI:

BIOS (old)

  • Uses MBR partition table
  • Bootloader installed into first 512 bytes of disk
  • Max disk size 2TB
  • Max 4 partitions

UEFI (modern; almost always used)

  • Uses GPT partition table
  • Needs an EFI partition (FAT32, ~512MB)
  • Max disk size 8zb
  • Max 128 partitions
MBR partitioning scheme (by default in Virtual-box unless you force GPT)

what is LVM

LVM (Logical volume manger) is flexible disk management tool for Linux. we have three layer:

PV (Physical Volume): Any partition or Disk it‚Äôs Physical Volume.

VG (Volume Group): Group of Physicals Volumes like sda1 and sda2 and sdb in Group1.

LV (Logical Volume): Partitions can have size of all Physicals Volumes from some Volume Group and can access to it from mount point.

LUKS stands for Linux Unified Key Setup, a standard for disk encryption on Linux that creates a secure, encrypted container on a block device, requires a passphrase to unlock the data for access

Sudo Policy

Config sudo policy while in /etc/sudoers.d/sudo_config file.

sudo is command line allow user to be superuser (aka root)

Set limited to attempt incorrect password when use sudo:

Defaults passwd_tries=3
Defaults                parameter,   parameter, ...     #affect all users on any host
Defaults@Host_List      parameter,   parameter, ...     #affects all users on a specific host
Defaults:User_List      parameter,   parameter, ...     #affects a specific user
Defaults!Cmnd_List      parameter,   parameter, ...     #affects  a specific command
Defaults>Runas_List     parameter,   parameter, ...     #affects commands being run as a specific user

Customize incorrect password massage:

Defaults badpass_message="Nice try lol"

Defaults logfile="/var/log/sudo/sudo.log"

Password Policy

Modify in file /etc/login.defs 

PASS_MAX_DAYS 30 #Maximum days of valid password 
PASS_MIN_DAYS 2  #Minimum days allowed between change password 
PASS_WARN_AGE 7  #Number of days before see worning a password expires

Install password quality liber

sudo apt install libpam-pwquality

Edit in file /etc/pam.d/common-password 

password requisite pam_pwquality.so retry=3 minlen=10 dcredit=-1 ucredit=-1 lcredit=-1 maxrepeat=3 reject_username difok=7 enforce_for_root
  • retry: how many time you can try made password with this rules
  • minlen: minimal of long the password
  • dcredit: how many digit in password when use minus - this mean minimal digit and when use plus + this mean maximal
  • ucredit: like dcredit but for uppercase
  • lcredit: and this for lowercase
  • maxrepeat: maximum characters can be consecutive
  • reject_username: cannot continue username in password
  • difok: minimal difference character between old password and new
  • enforce_for_root: apply this rules for sudo

create file monitoring.sh:

cpuf=$(nproc)
cpuv=$(cat /proc/cpuinfo | grep processor | wc -l)
ram=$(free -m | awk 'NR==2{printf "Memory: %s/%sMiB (%.2f%%)\n", $3, $2, ($3*100)/$2}')
disk=$(df -h / | awk 'NR==2{printf "Disk: %s/%s (%s) \n", $3, $2, $5}')
cpul=$(top -bn2 | awk 'NR==8 {printf "CPU Load: %s%% \n", $9}')
lb=$(who -b | awk '{printf "Last boot: %s %s \n", $3, $4}')
lvmu=$(if [ $(lsblk | grep "lvm" | wc -l ) -gt 0 ]; then echo yes; else echo no;fi | awk '{printf "LVM: %s \n", $1}')
tcpc=$(ss -ta | grep ESTAB | wc -l | awk '{printf "Connections TCP: %s ESTABLISHED\n", $1}')
ulog=$(printf "User log: " && who | awk '{print $1}' | sort -u | wc -w)
ip=$(ip -4 addr | grep inet | awk 'NR==2{printf "Nerword IP: %s", $2}' | cut -d/ -f1 | tr -d '\n')
mac=$(ip addr | grep link/ether | awk '{printf " (%s) \n", $2}')
cmd=$(journalctl _COMM=sudo -q | grep COMMAND | wc -l | awk '{printf "Sudo: %s cmd\n", $1}')
